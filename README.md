# demoMicrofrontend

This is a Demo about how we can use Micro Frontends to have idependent apps for each content of our global app.


# Getting started

To run the project go to each directory:

1. `cd content`
2. `yarn install`
3. `yarn build`
4. `yarn start`

5. `cd ../book-order`
6. `yarn install` 
7. `yarn start`

8. `cd ../browse`
9. `yarn install`
10. `yarn start`

11. `cd ../container`
12. `yarn install`
13. `yarn start`
